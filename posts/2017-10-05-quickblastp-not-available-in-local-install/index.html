quickBLASTP not available in local install

Oct 5, 2017 by Owen

I recently read about “QuickBLASTP, an accelerated version of BLASTP” that is now available in the online BLAST application.    There is a question on the NCBI announcement page, asking “Does exits the possibility of downloading the corresponding executables so we can try it locally?” but no answer is provided.

I sent this question to the help desk and got a disappointing reply.

Question to NCBI help desk:

How to run QuickBLASTP on a local installation


The announcement of QuickBLASTP is very short and doesn’t include many details. 

Is it possible to replicate “QuickBLASTP” on a local installation? Is it just blastp run with certain options? Does it require any particular version of BLAST+? 

Thank you

Reply

Hello,


Thank you for your interest. We currently have no QuickBLAST option for standalone blast+. It uses a k-mer based seeding approach, so it’s very different from blastp.

Best regards, [redacted]
