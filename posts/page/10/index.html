Convert FASTQ to FASTA using sed command

Sep 4, 2012 by Owen
Easiest possible solution, and I believe significantly faster than awk-based methods: sed -n '1~4s/^@/>/p;2~4p' The only assumption made is that each read occupies exactly 4 lines in the FASTQ file, but that seems pretty safe, in my experience. If you find this helpful, go up-vote my answer in StackOverflow.

lftp mirror: best way to download an entire directory from a FTP server

Apr 4, 2012 by Owen
As much as I hate FTP, there situations where one must use it. I finally have settled on the best way to get an entire directory, with recursive file retrieval: lftp -e “mirror dir1/subdir2/subdir3” -u username,password ftphost.domain.com Of course, FTP must die, but until then, I am glad there is lftp.
